總共有$n$件衣服，各自有不同的價格$(a_1, a_2, ..., a_n)$。\\
一開始第$i$件會在編號為$i$的桶子裡，接下來會有$m$筆操作\\
每次操作有三種選擇：\\
- A x y ：將編號第$x$的衣服所在桶子裡面所有的衣服，倒到第$y$件衣服所在的桶子 $(1 ≤ x, y ≤ n)$\\
- M x y ：第$x$件衣服的售價改為$y (1 ≤ x ≤ n, 1 ≤ y ≤ 10^9)$\\
- Q x y ：查詢第$x$件衣服所在的桶子，裡面第$y$大的售價 $(1 ≤ x ≤ n)$\\
\\
$1 ≤ n ≤ 10^5$\\
$1 ≤ m ≤ 4 * 10^5$\\
